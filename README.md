# Repositorio de Proyectos del Curso de Programación Full Stack - Instituto EGG

Este repositorio contiene todos los proyectos creados durante el curso de Programación Full Stack dictado por el instituto EGG a lo largo del año 2022. Aquí se reúne todo lo que fui creando durante mi primera experiencia en el desarrollo de software, consolidando ejercicios y proyectos en un único lugar.

## Contenido

### Frontend-HTML
Enfocado en la introducción al desarrollo web utilizando HTML. Este directorio incluye ejercicios para aprender la estructura y diseño básico de páginas web.

### Frontend-JavaScript
Incluye ejercicios y proyectos diseñados para aprender y practicar JavaScript, abordando desde conceptos básicos hasta aplicaciones más interactivas.

### FullStack-SpringSecurity
Este proyecto de backend incluye carpetas enfocadas en controladores, servicios y repositorios, lo cual permite gestionar datos y establecer seguridad en aplicaciones web usando Spring Security. Además, se exploran métodos de autenticación y se realiza un manejo detallado de entidades Java.

### Backend Collections
Contiene varios ejercicios que abordan diferentes funcionalidades:
- Ej1: Implementación de servicios.
- Ej3: Gestión de alumnos y sus notas.
- Ej4: Manejo de películas y ordenación.
- Ej5: Registro y búsqueda de países.
- Ej6: Gestión de productos y operaciones.
- Extra1: Operaciones con listas de números.

### Backend JDBC
Incluye clases dedicadas a la conexión y manipulación de bases de datos utilizando JDBC. Ejemplo:
- DAO: Clase abstracta que encapsula métodos genéricos como conectar, desconectar y ejecutar consultas.

### Backend JPA
Proyectos que emplean JPA para la persistencia de datos:
- NoticiaRepositorio.java: Proporciona consultas personalizadas para manejar entidades Noticia.
- UsuarioRepositorio.java: Gestión de datos de usuarios mediante consultas específicas.
- AutorRepositorio.java: Permite buscar autores por nombre utilizando consultas JPA.

### Backend Herencia
Explora conceptos de herencia en Java:
- Ej1: Modelos de animales y su comportamiento alimenticio.
- Ej2: Creación y procesamiento de datos de electrodomésticos.
- Ej3: Manejo avanzado de electrodomésticos y cálculo de precios.
- Ej4: Cálculo de figuras geométricas como círculos y rectángulos.
